# v5 戦略ドキュメント

## エグゼクティブサマリー

v4が過去最高スコア（4.12）を達成。v5では**データセット品質の改善**によりさらなるスコア向上を目指す。

### v4からの主要な学び

| 指標 | v4結果 | 残課題 |
|------|--------|--------|
| 総合スコア | **4.12** | さらなる向上を目指す |
| ALFWorld成功率 | **54%** | task limit reached (38%) が主な失敗要因 |
| Invalid action | **8%** | ほぼ解消 |
| DBBench accuracy | **53.1%** | 維持・向上 |

---

## 1. v5の戦略方針

### 1.1 メイン戦略: データセット品質改善

v4では学習率調整（1e-6）でInvalid action問題を解消。
v5ではデータセット品質を改善し、**task limit reached (38%)** の削減を目指す。

```
v4の成功要因: 学習率 2e-6 → 1e-6
v5の改善点: 高品質データセット（v6）の作成
```

### 1.2 戦略の根拠

[`analysis/invalid_action_analysis.md`](analysis/invalid_action_analysis.md) で特定した問題:

1. **失敗トラジェクトリーの混入**: 300件以上の失敗パターンが学習データに含まれる
2. **"Nothing happens."パターン**: 234件の無効応答が学習に含まれる

これらを除去することで、モデルはより効率的なタスク解決パターンを学習できる。

## 3. ALFWorld v6 データセット作成要件

### 3.1 データクリーニング要件

| # | 要件 | 優先度 | 期待効果 |
|---|------|--------|----------|
| 1 | 失敗トラジェクトリーの除外 | **高** | 失敗パターンの学習を防ぐ |
| 2 | "Nothing happens."を含むターンの除外 | **高** | 無効アクションの学習を防ぐ |
| 3 | 成功トラジェクトリーのみ選択 | **高** | 成功パターンの強化学習 |
| 4 | 効率的なタスク完了パターンの選択 | 中 | task limit reached削減 |

### 3.2 v6データセット作成スクリプト要件

```python
# 概念的なフィルタリングロジック
def filter_trajectory(trajectory):
    # 1. 失敗トラジェクトリーを除外
    if trajectory.get("trajectory_outcome") == "failure":
        return False

    # 2. "Nothing happens."を含むトラジェクトリーを除外
    for turn in trajectory["turns"]:
        if "Nothing happens." in turn.get("output", ""):
            return False

    # 3. 成功かつ効率的なトラジェクトリーのみ
    if trajectory.get("trajectory_outcome") == "success":
        return True

    return False
```

### 3.3 期待されるデータセットサイズ

| データセット | v5 (現行) | v6 (予想) |
|-------------|-----------|-----------|
| 総サンプル数 | ~3000 | ~1500-2000 |
| 失敗トラジェクトリー | 300+ | **0** |
| "Nothing happens." | 234 | **0** |

## 5. リスク分析と緩和策

### 5.1 リスク

| リスク | 可能性 | 影響 | 緩和策 |
|--------|--------|------|--------|
| データセットサイズ減少による学習不足 | 中 | 中 | エポック数を3に増加検討 |
| 過度なクリーニングによる多様性喪失 | 低 | 中 | 段階的クリーニング |
| v4からの性能低下 | 低 | 高 | v4設定を維持しつつデータのみ変更 |

### 5.2 フォールバックプラン

v5で性能低下した場合:
- v5a: エポック数を3に増加
- v5b: データクリーニングを緩和（失敗トラジェクトリーのみ除外）
